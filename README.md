# Devskills
### Primeros pasos
- Clonar repositorio
- Ejecutar npm run serve
- Entrar a la carpeta front
- Ejecutar npm run serve en dicha carpeta
## Instalar dependencias
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
